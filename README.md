## Projects

**[corp-local-soc-lab](https://github.com/charifkati/corp-local-soc-lab)**
Personal SOC lab across 3 VMs (Windows Server 2022, Kali Linux, Ubuntu + Splunk). Simulated 5 Active Directory attacks mapped to MITRE ATT&CK including Kerberoasting and DCSync, wrote SPL detection rules that exposed two blind spots in the default Windows audit policy, and documented findings in structured incident reports.
